# jekyll-Blog
Personal Blog

#### Repository
> git@github.com:videwang/jekyll-Blog.git

### Quickstart   
1. Get codes   
`$ git clone <repository>`    
2. Init Ruby Project   
Go into <repository> folder run:   
`$ bundle init`
3. Add jekyll to Dependency   
`Gemfile` has been generated then:  
Add `gem "jekyll"` to `Gemfile` file    
4. Install jekyll   
`$ bundle install`   
5. Running locally   
`$ bundle exec jekyll serve`   
6. Ready to play
